Detailed Description

template<board::AnalogPin BG = board::AnalogPin::BANDGAP>
class analog::PowerVoltage< BG >

API that uses bandgap feature to calculate current voltage fed to the MCU.

Template Parameters
BGa unique bandgap analog pin for the MCU target; although the accepted type is board::AnalogPin enum, only bandgap inputs are allowed otherwise a compilation error will occur.
