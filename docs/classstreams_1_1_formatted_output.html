template<typename STREAM>
void streams::FormattedOutput< STREAM >::puts ( const flash::FlashStorage *  str)
inline

Append a string, stored on flash memory, to the buffer.

The `'\0'end character of @p str is not transmitted. If the buffer is full, thenon_overflow()will be called. Once all string content has been appended,on_put()` will be called, even if an overflow has occurred. Example:

output.puts(F("Hello, World!\n"));
