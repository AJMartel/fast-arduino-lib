Detailed Description

Defines all digital output pins of ATtinyX5, usable as direct external interrupt pins.

Each symbol is in the form Dxx_Pyz_EXTu, where xx is the imaginary pin number , y is the port letter (B, C or D), z is the bit number for that pin within its port and u is the number of the interrupt for that pin. This namespace exists for the sole purpose of quickly finding an external interrupt pin among all digital IO pins.
