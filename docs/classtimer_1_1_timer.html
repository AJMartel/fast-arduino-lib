Detailed Description

template<board::Timer TIMER>
class timer::Timer< TIMER >

General API to handle an AVR timer.

Note that many timer usages will require ISR registration with one of the macros defined in this header file.

void timer::Timer< TIMER >::begin ( TIMER_PRESCALER  prescaler)
inline

Start this timer in the currently selected mode, with the provided prescaler value.

This method does not enable timer interrupts, hence no ISR registration is needed. Note that this method is synchronized, i.e. it disables interrupts during its call and restores interrupts on return. If you do not need synchronization, then you should better use _begin() instead.
