Detailed Description

template<board::USART USART>
class serial::hard::UATX< USART >

Hardware serial transmitter API.

For this API to be fully functional, you must register the right ISR in your program, through REGISTER_UATX_ISR(), then call register_handler() immediately after the UATX instance has been constructed.
