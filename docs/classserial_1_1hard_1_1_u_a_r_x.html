Detailed Description

template<board::USART USART_>
class serial::hard::UARX< USART_ >

Hardware serial receiver API.

For this API to be fully functional, you must register the right ISR in your program, through REGISTER_UARX_ISR(), then call register_handler() immediately after the UARX instance has been constructed.
